GameMath Namespace
GameMath is a small, free open-source math library for games.
Classes
  ClassDescription
Angle
Common math operations on angles in radians.
BallisticTrajectory
Path that a thrown or launched projectile or missile will take under the action of gravity, neglecting all other forces, such as friction from aerodynamic drag, without propulsion.
BoxIntersection
Intersection tests for boxes.
Bresenham
Optimized version of the Bresenham line algorithm that finds an approximation to a straight line between two given points.

Random2
Implementation of the Ranq1 struct found in Numerical Recipes in C: 3rd Edition. Combined generator (Ranq1 = D1(A1(right-shift first)) with a period of 1.8 x 10^19.
